Step-by-Step Build Guide
01

Brand Setup — KUVEXO CORE Identity (Days 1–3)

Finalize app name (GharOS recommended). Create logo in Figma using hex-shield concept. Export 512×512 PNG. Register domain (kuvexocore.com or gharo.app). Set up brand color variables.

02

Design All Screens in Figma (Days 4–10)

Create 375×812px frames (iPhone size — Flutter will adapt). Design Splash → Onboarding → Home → all 8 feature screens → Settings. Use the Dark Cosmos color palette. Export assets.

03

FlutterFlow Project Setup (Days 11–14)

New project at flutterflow.io. Import Figma via FlutterFlow Figma plugin OR rebuild manually (faster). Set theme colors, fonts (Orbitron + Sora + DM Sans). Create reusable components: feature card, button, header.

04

Splash Screen + Onboarding (Days 15–17)

Add Lottie animation to splash (ring-pulse from LottieFiles). Set 2.5s timer → navigate to Home. Build 3 onboarding slides with swipe. Store user name in Hive on completion. Add "Skip" logic.

05

Core Features — Bill + Medicine + Docs (Days 18–28)

Build CRUD screens: Add → List → Delete for each. Connect to Hive local DB. Set up local push notifications (flutter_local_notifications package). Test reminders on real device. Color urgency logic.

06

Tasks + Grocery + Notices (Days 29–35)

Task list with per-member tabs. Grocery with WhatsApp share (url_launcher package). School notices with camera capture (image_picker). Store photos in local app folder.

07

Nearby Services — Google Maps Integration (Days 36–40)

Enable Google Maps Places API in Google Console. Add API key to FlutterFlow. Build search UI: user taps category → app searches "plumber near [GPS]" → shows card list with call button.

08

Testing — Real Phone Testing (Days 41–46)

Install on 3 different Android phones (₹5K, ₹15K, ₹30K range). Ask 1 elderly person to test without guidance — watch where they get confused. Fix those points. Test all notifications. Test on Hindi language.

09

Play Store Launch Prep (Days 47–52)

Create store graphics (feature image 1024×500, screenshots 1080×1920). Write description in Hindi + English. Generate Privacy Policy free at termsfeed.com. Submit APK to Google Play Console ($25 one-time).

10

Launch + Marketing (Day 53+)

Share in 20 WhatsApp family groups. Make 2 YouTube Shorts showing "before/after" — messy paper reminders vs GharOS. Post on Facebook Indian parenting / housing groups. Ask first 50 users for Play Store review.
